# Toss the coin

At the point where you have two options and have to choose one, tossing the coin helps a lot.
Nowadays when electronic means of transaction are gaining popularity, coins are disappearing from pockets.
This ``` toss the coin ``` can help in such a situation.
It has a green background, with options to assign names before toss.
It has real images of the coin to make this look more realistic.


### Main features:
    * Responsive design for different screen sizes.
    * Good colour combination.
    * Options to assign names for HEADS and TAILS.
    * Images of real coins for a more realistic feel.
